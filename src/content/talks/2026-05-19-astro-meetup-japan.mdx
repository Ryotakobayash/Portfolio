## コンセプト：ブログ × ダッシュボード

- 普通にブログを作るだけだと物足りない、1芸ほしい
- 仕事で触っているグラフ表示系のライブラリを使いたい（ついでに詳しくなりたい）
- データソースとして GA4 / GitHub が触れるのも Good

→ ブログ × ダッシュボードの個人サイトを作ろう、と決めました。

## 全体の構成

<TwoColumn>
<div>

### Astro が主役のところ
- **Pages / Layouts** — `src/pages` でルーティング
- **Content Collections** — `posts` / `talks` を MDX 管理
- **静的に出せるもの** — 一覧, SEO, OGP, RSS

</div>
<div>

### React を island 化したところ
- `client:load` / `client:only="react"` で島化
- Highcharts で記事 PV / GitHub Activity
- 動くもの・重いものだけクライアントに

</div>
</TwoColumn>

> ブログとしてのみやすさ・軽量さを保ちつつ、ダッシュボードも成立。

<Notes>
- island の話はここで一言だけ
- 「主役は Astro / 必要なところだけ React」の構図を強調
</Notes>
</Slide>
